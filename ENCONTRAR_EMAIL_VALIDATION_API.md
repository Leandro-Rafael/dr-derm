# Como Encontrar Email Validation API na Abstract API

## Problema: Não aparece na lista

A "Email Validation API" pode não aparecer no menu mobile. Tente estas opções:

## Opção 1: Acessar pelo Desktop

1. **Abra no computador** (não no celular):
   - Acesse: https://app.abstractapi.com/

2. **Procure por "Email Validation"**:
   - No menu lateral, pode estar como:
     - "Email Validation"
     - "Email Verification" 
     - "Verify Email"

3. **Ou use a busca**:
   - No topo, há uma barra de busca
   - Digite: "email validation"

## Opção 2: Acessar Diretamente

Tente acessar diretamente:
- https://app.abstractapi.com/api/email-validation
- https://www.abstractapi.com/api/email-validation

## Opção 3: Verificar na Página Principal

1. Acesse: https://www.abstractapi.com/api/email-validation
2. Clique em **"Get Started"** ou **"Open Dashboard"**
3. Deve redirecionar para a API

## Opção 4: Verificar em "All APIs"

1. No dashboard, procure por **"All APIs"** ou **"Browse APIs"**
2. Procure na lista completa
3. Pode estar em uma seção diferente

## Se Ainda Não Encontrar

A Email Validation API pode não estar disponível no seu plano atual. Alternativas:

### Alternativa 1: Usar sem API (Verificação Básica)
- O código já funciona sem API
- Faz verificação de formato e domínios conhecidos
- Menos rigoroso, mas funciona

### Alternativa 2: Outra API Gratuita
- **MailboxValidator** (100 verificações/mês grátis)
- **EmailListVerify** (verificações gratuitas)
- **Hunter.io** (25 verificações/mês grátis)

### Alternativa 3: Usar Email Reputation API
- Você viu "Email Reputation" na lista
- Essa API é diferente (verifica reputação, não validade)
- Não serve para validar se email existe

## O Que Fazer Agora

**Recomendação:** Por enquanto, use o sistema sem API (já está configurado). Ele vai:
- ✅ Verificar formato do email
- ✅ Verificar domínios conhecidos (Gmail, Yahoo, etc.)
- ✅ Aceitar emails com formato válido

Depois você pode adicionar API quando encontrar.
